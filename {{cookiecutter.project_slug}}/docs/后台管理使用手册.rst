后台管理使用手册
=================


使用对象
-----------

.. glossary::

    超级管理员
        超级管理员拥有所有权限。
        超级管理员负责创建公司、分配权限、设定座席登录密码等初始化工作

    组长
        组长负责从运营商系统同步项目，然后把呼叫清单分配给具体的座席人员。组长同时可以在这里查看总体的外呼情况。


数据结构
----------

.. glossary::

    公司
        系统的权限是按公司隔离的. 每个公司员工只能看到自己公司的数据.
